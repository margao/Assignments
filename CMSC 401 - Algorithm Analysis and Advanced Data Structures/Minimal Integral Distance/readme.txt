cmsc401.java

Description:
-----------------------
Takes integers from standard input (System.in) where first line of input
contains the number of n positive integers, where 2 < n < 1000000.
Second line of input contains all n positive integers x_1, x_2,..., x_n where
x_i is an element of all natural numbers and 0 < x_i <= 1000000000 for all i in [n].
Outputs single line with two positive integers that have the minimal integral distance.


Compile command: javac cmsc401.java
Run command: java cmsc401

Example: 
Input number of n positive integers:
10
Input data content of array separated by space:
6 2 12 14 91 18 5 333 32 41

Minimum integral distance: 
5 6
